# 計算工具箱 · Site Calc Toolbox

工地與設計案頭常用的計算小工具。即算即檢核，附法規與規範條文出處，算完可直接複製成檢核紀錄。

**線上使用**：https://radihuang.github.io/calc-toolbox/

零建置（zero-build）：每個工具都是單一 `index.html`，React 18 + Babel 由 CDN 載入、瀏覽器端即時編譯。開檔即用，GitHub Pages 直接部署。

## 結構

```
index.html          入口頁（純 HTML/CSS）
slope/index.html    坡度計算機
rebar/index.html    鋼筋混凝土計算機
steel/index.html    鋼構重量計算機
fence/index.html    北北桃施工圍籬面積計算機
```

## 工具

### 坡度計算機 `slope/`
長度與前後高差互算坡度（%）、斜率（1:n）、‰、角度；反算模式由目標坡度反推高差或水平長。依台灣現行法規即時提醒：

- 洩水坡度 — 建築技術規則建築設備編（常用 1/100～1/50）
- 車道坡道 — 建築技術規則建築設計施工編 §61（≤ 1:6）
- 無障礙坡道 — 建築物無障礙設施設計規範 206（≤ 1/12，含高差放寬）

### 鋼筋混凝土計算機 `rebar/`
輸入鋼筋號數（D10–D36）、間距、構材尺寸，即時計算鋼筋量 As、鋼筋比 ρ、有效深度 d 與每米（每 m²）鋼筋重量、混凝土體積。版／梁／柱三種模式逐項檢核，各卡片註明條文出處，可一鍵複製為檢核紀錄：

| 檢核項目 | 依據 |
|---|---|
| 保護層厚度（依曝露條件） | 表20.5.1.3.1 |
| 版最少鋼筋量（收縮溫度） | §7.6.1.1、§24.4.3.2 |
| 版主筋／溫度筋最大間距 | §7.7.2.3、§24.4.3.3 |
| 梁最少受拉鋼筋 As,min | §9.6.1.2 |
| 梁箍筋間距 | §9.7.6.2.2 |
| 柱主筋鋼筋比 0.01–0.08 | §10.6.1.1（耐震另見 §18.4.3.1） |
| 柱橫箍筋間距 | §25.7.2.1 |
| 鋼筋最小淨間距 | §25.2.1（柱 §25.2.3） |

條號依內政部《建築物混凝土結構設計規範》（112.8.10 修正發布，自 113.1.1 生效；章節架構對應 ACI 318-19）。鋼筋標稱直徑、斷面積、單位重依 CNS 560。材料強度採規範主列之公制單位（kgf/cm²）。

另附牌價欄位（鋼筋 元/噸、混凝土 元/m³，預設 20000／3000），依計算所得鋼筋重量與混凝土體積自動概估費用，並附鋼筋、混凝土行情參考網站連結；牌價僅供概估，實際報價以供應商當日報價為準。

### 鋼構重量計算機 `steel/`
輸入斷面尺寸與長度，即時計算單位重（kg/m）、總重（kg／t），附斷面示意圖，可一鍵複製估算紀錄。支援三種型式：

- **TS 方管** — 外寬 × 外高 × 壁厚
- **BOX 矩管** — 外寬 × 外高 × 壁厚，可加內隔板（橫隔板）
- **BH 組合 H 型鋼** — 高 H × 寬 B × 腹板厚 tw × 翼板厚 tf

單位重公式（鋼材密度統一 7.85 kg/dm³）：

| 型式 | 斷面積 |
|---|---|
| TS／BOX | 〔(W＋H)×2 − 4t〕× t |
| BH | 2·B·tf ＋ (H − 2tf)·tw |
| 內隔板（每片） | 內寬 × 內高 × 板厚 |

斷面以尖角近似、不計圓角，實際型鋼因含圓角略輕，此值為快速估算。內隔板為填滿內淨斷面之橫隔板，設間距則依構材長度自動推算片數，否則採指定片數。內附 STK／STKR／STKN／SM／SN／BCR／BCP 等鋼種選單，僅供規格標示，不影響重量。

### 北北桃施工圍籬面積計算機 `fence/`
選擇適用城市（臺北／新北／桃園），按基地各面輸入長度、臨接道路寬度與版片高度（2.4／3／4m，可增減面數以因應不規則基地），即時算出各項面積數量並附立面示意圖，可一鍵複製估算紀錄：

- **圍籬版片** — 各面長×高，依高度（2.4／3／4m）分組加總
- **RC 防溢座** — 各面長×防溢座高（圍籬 ≥3m 取 60cm、否則 30cm）
- **綠化／美化面積** — 路寬達門檻之沿街面，扣大門後依比例分為密植綠化與彩繪美化
- **安全走廊頂蓋** — 路寬達門檻且面長 ≥10m 各面之長×走廊寬
- **透空圍籬、隔音／防塵帆布** — 另依輸入計

三市法規門檻（綠美化、安全走廊）已預填但可於欄位覆寫：

| 項目 | 臺北市 | 新北市 | 桃園市 |
|---|---|---|---|
| 圍籬高度 | ≥2.4m（走廊側 ≥3m） | ≥2.4m（鋼板 ≥1.2mm） | ≥2.4m |
| 綠美化門檻 | 臨接 ≥10m 道路/公園/綠地/廣場 → 沿街面 ≥½ 密植綠化 | 依圍籬綠美化推動實施計畫（實務多以沿街面 ½ 計） | 臨都市計畫道路或 ≥8m 道路 → 美化 ≥沿街面 ½ |
| 安全走廊門檻 | 臨計畫道路人行道即檢討 | 面前道路 ≥12m 且臨接長 ≥10m | 依個案 |
| 防溢座（<3m／≥3m） | 30×15／60×30 cm | 30×15／60×30 cm | 法規未明列，沿用 30/60 |

依據：《臺北市建築物施工中妨礙交通及公共安全改善方案》、《新北市政府建築工程施工管理要點》、《桃園市建築物施工中管制要點》。門檻與尺寸為現行條文整理，實際以各市最新法規、核定圖說與都市設計審議認定為準。

## 部署

GitHub Pages（根目錄）即可，目前部署於 https://radihuang.github.io/calc-toolbox/ 。若沿用既有 `slope-calculator` repo：

1. 將原 `index.html` 移至 `slope/index.html`
2. 加入本專案之 `index.html`（入口頁）與 `rebar/index.html`
3. 原網址即成為工具箱入口；坡度計算機改為 `…/slope/`

## 免責

各工具之法規數值為現行常見規定之整理，僅供快速研判與紀錄。淨間距檢核未計入粗粒料 4/3 倍粒徑條件；箍筋重量為含彎鉤概估；鋼構重量以尖角斷面近似、不計圓角。實際設計、耐震細部、下料計價與查驗請以最新條文、廠商規格表、簽證文件與主管機關認定為準。
